# UMAP Article

```bash
yarn
yarn dev

# To run individual figures standalone
yarn dev:toy
yarn dev:toy_comparison
yarn dev:cech
yarn dev:hyperparameters
yarn dev:mammoth-umap
yarn dev:mammoth-tsne
```
